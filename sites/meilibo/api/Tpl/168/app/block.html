封号规则
  • 一、适用范围：
  • 适用于所有主播及用户
  • 二、具体封号规则：
  • 1．严禁发表反党反政府的言论，或做出侮辱诋毁党和国家的行为
  • 2．严禁直接或间接传播淫秽、色情信息、或进行淫秽、色情表演
  • 3．严禁丑陋、低俗、性感、引诱、挑逗性内容，着装暴露、性暗示、引诱用户付费等
  • 4．严禁通过任何方式传播毒品、展示枪支、管制刀具等
  • 5．严禁通过任何方式宣传或展示打牌、赌博等
  • 6．严禁冒犯性、羞辱性、歧视性的语言和行为
  • 7．严禁进行各类广告宣传、恶意发布广告的行为
  • 8．严禁进行其他涉政、涉黄、违法、侵权及其他喵榜直播相关秩序规范内容
  • 9．严禁直播过程中吸烟，或展示暴力内容
  • 10．严禁拍摄、转播非喵榜直播平台等其他直播内容
  • 11．禁止直播睡觉、挂机、黑屏、画面定格等行为
  • 12．禁止直播电影、电视节目、电子游戏等内容
  • 13．不得通过非官方渠道购买喵榜直播钻石
  • 三、处理方法：
  • 1．喵榜直播对以上行为实施从严标准，所有主播或用户，如果言论和行为有违反上述内容的，将视情节轻重予以1天、30天或永久封号处理
  • 2．任何用户都可使用“举报”按钮对违规主播、用户，如果发现用户的举报投诉是恶意行为，我们也将对恶意举报，投诉的用户进行相应的处罚
